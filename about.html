About Us

Radiant Utilities is a Discord bot created for the Radiant Valorant Discord server. Its primary function is to host custom games moderated by server administrators, enabling players to connect and compete in a safe, enjoyable environment.

Radiant Utilities also includes moderating and managing tools for administrators to monitor user activity, manage channels, and enforce rules.

The bot is exclusively available for the Radiant Valorant server, ensuring a consistent, high-quality experience. With its user-friendly interface and robust feature set, Radiant Utilities is a must-have tool for serious Valorant players.
